# Blog con IA

Proyecto basico con nodejs de una api para un blog hecho con express, con pruebas de controlador con jest, documentacion con swagger y docker.

## TECNOLOGIAS

- [Nodejs]
- [Express]
- [Joi]
- [Faker]
- [Factory-Girl]
- [Mongoose]
- [Nodemailer]
- [Jest]
- [Swagger]
- [Docker]

## VARIABLES DE ENTORNO

- API_URL = /api/v1
- HOST_URL = 'http://localhost:8000/'
- SECRET = _tu secret password_
- CONNECTION_LOCAL = _url de atlas_
- EMAIL_PORT = _puerto de la app de mensajeria_
- HOST_EMAIL = _host de la app de mensajeria_
- EMAIL_USER = _usuario de la app de mensajeria_
- EMAIL_PASSWORD = _password de la app de mensajeria_
